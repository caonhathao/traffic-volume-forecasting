# Dự án thử nghiệm: Traffic Volume Forecasting
Demo Streamlit dự báo lưu lượng xe (Metro Traffic Volume) thời gian thực. Áp dụng Sliding Window xử lý Stream Data tuần hoàn, dùng thuật toán dự báo (Moving Average/ML) nhận diện kẹt xe và đưa ra cảnh báo ùn tắc ngay trên biểu đồ động.

# Dữ liệu sử dụng
- Dataset: [Metro Interstate Traffic Volume](https://www.kaggle.com/datasets/pooriamst/metro-interstate-traffic-volume)

# Mục tiêu dự án
- Dự án được xây dựng để dự đoán lưu lượng xe theo thời gian thực.
- Cho phép đưa ra các thông báo điều hướng lưu lượng phương tiện trong các trường hợp đặc biệt.
